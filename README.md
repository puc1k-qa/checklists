# Чек лист на проверку веб-приложения

##  UI / UX
- Все элементы отображаются корректно
- Нет наложений элементов друг на друга
- Шрифты читаемы и одинаковы по стилю
- Кнопки имеют понятные названия
- Ошибки отображаются пользователю понятно

---

##  Функциональное тестирование
- Регистрация работает корректно
- Логин/логаут работает
- Валидация полей работает
- Ссылки ведут на правильные страницы
- Формы отправляются корректно

---

##  Нефункциональное тестирование
- Проверка пустых полей
- Проверка некорректного email
- Проверка слишком длинных значений
- Проверка SQL injection (базовая защита)
- Проверка XSS (ввод скриптов)

---

##  API / Backend
- API возвращает корректный статус код
- Проверка response body
- Проверка обязательных полей
- Проверка ошибок (400/401/500)

-  Chrome

##  Тестирование производительности

##  Тестирование безопасности
- Пароли скрыты (••••)
